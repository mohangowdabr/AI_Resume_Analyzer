# 🚀 AI Resume Analyzer

An AI-powered Resume Analyzer that evaluates ATS (Applicant Tracking System) compatibility, identifies skills, highlights missing keywords, and generates personalized improvement suggestions using Google's Gemini AI.

## 📌 Features

* 📄 Upload Resume (PDF)
* 🤖 AI-Powered Resume Analysis
* 📊 ATS Compatibility Score
* 🛠️ Skill Detection
* 🔍 Missing Skills Identification
* 💡 Resume Improvement Suggestions
* 📈 Resume Strengths & Weaknesses Analysis
* 🌙 Dark Mode Support (Future Enhancement)
* 📥 Download Analysis Report (Future Enhancement)
* 📱 Responsive User Interface
